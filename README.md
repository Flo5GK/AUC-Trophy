# AUC-Trophy
--- 
AUC Trophy permet de créer des items minecraft, sous le format "Trophée" de AUC:


## Installation :
Simplement télécharger la dernière [release](https://github.com/Flo5GK/AUC-Trophy/releases) et lancer "Trophy AUC.exe"
